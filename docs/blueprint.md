# **App Name**: Zatoka Getaway

## Core Features:

- Homepage: Display key information about the hotel, attractive imagery and direct booking links
- About Us: Provide hotel history, mission, amenities, staff introductions, and values, with integrated SEO optimization for increased online visibility.
- Booking Page: Enable users to view room options, check availability, and book rooms. Incorporate real-time updates using rtk query and direct link to the booking engine.
- SEO Optimization: Implement SEO practices across the site for better search engine rankings. Focus on relevant keywords and meta descriptions for Zatoka, Odessa region.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) evoking the sea.
- Background color: Light cyan (#E1F5FE), a desaturated version of the primary color, complementing the marine theme, almost white, keeping a light feel.
- Accent color: Sandy yellow (#FFCA28), an analogous color providing a bright and sunny feel.
- Font pairing: 'Playfair' (serif) for headings to convey elegance, and 'PT Sans' (sans-serif) for body text for clarity.
- Use minimalist icons to represent amenities and services, echoing the coastal theme (e.g., waves, sun, room keys).
- A clean, spacious layout with high-quality images of the hotel and Zatoka to enhance the visual appeal.
- Subtle transitions and animations (e.g., page loading, image reveals) to enhance user experience.
